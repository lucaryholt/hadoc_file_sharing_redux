HADOC File Sharing

Quick, easy and anonymous file sharing!

Why?

This was made as a school project leaning Node.JS.
Visit the Github repository here.

Privacy

This service does not save any info about you, unless you sign up or write a receiver email address.
The receiver email is only saved as long as the files are still active.
Once files expire all data regarding them are deleted.

Cookies

Only the most important cookies are saved, to keep you logged in.
